# management_system

这是个人学习vue的第三个实战项目，结合vue全家桶和elementUI库做的一个电商后台管理系统。

### 项目运行

##### 1.安装依赖

```
npm install
```

##### 2.运行项目
```
npm run serve
```



### 技术栈

1.Vue全家桶(vue2.x,vue-cli,vue-router)

2.axios

3.Webpack

4..Es5及以上

5.Element-UI

## 项目实现功能：

### 登录模块：

1. ##### 表单数据校验

2. ##### 重置表单按钮

3. ##### 登录成功后将token值保存到sessionStorage中

   

###  首页模块：            

- #### 用户管理

  1. 搜索用户功能
  2. 添加用户按钮
  3. 修改用户信息按钮
  4. 删除用户信息按钮
  5. 分配用户角色按钮
  6. 底部分页栏
  7. 顶部面包屑导航栏

- #### 权限管理

  - #### 角色列表

    1.  添加角色按钮
    2. 编辑角色信息按钮
    3. 删除角色信息按钮
    4. 分配权限按钮
    5. 顶部面包屑导航栏

  - #### 权限列表

    1.  权限列表的渲染
    2. 顶部面包屑导航栏

- #### 订单管理

  - ####  商品分类

    1. 添加分类按钮
    2. 编辑分类按钮
    3. 删除分类按钮
    4. 底部分页栏
    5. 顶部面包屑导航栏
